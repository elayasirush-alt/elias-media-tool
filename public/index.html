Search media from different sites

Search anything and get results from every selected source. Each result has Open, Media file, and Download where available. Canva and search-term generation are removed.

Every selected source will appear. If an API key is missing or there are no direct results, the tool shows an external source-search link.

Timestamp media suggestions

Paste timestamped text. The tool will suggest the kind of media to use for each timestamp and can pull example results from all selected sources.

For long scripts, media examples are limited to the first scenes so Render does not timeout.
